# React News App

A simple and responsive news application built with **React**.  
It uses the **NewsAPI** to fetch live articles and lets users browse by category, search topics, and switch between English/Hindi.

---

## Features
- Keyword based search  
- Language toggle (English/Hindi)  
- Category routes: Education, Entertainment, Sports, Science, Health, Technology, Politics, Bollywood, Hollywood, Jokes, Crime  
- Infinite scroll for smooth loading  
- Responsive UI with Bootstrap  
- API key handled securely via `.env.local` and `.gitignore`  

---

## Tech Stack
- React.js (functional components + hooks)  
- React Router v6  
- Bootstrap 5  
- react-infinite-scroll-component  
- NewsAPI.org
